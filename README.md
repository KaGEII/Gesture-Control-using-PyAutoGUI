# Gesture-Control-using-PyAutoGUI
This is a program to control the device using the hand gestures captured through your computer's webcam. With different gestures, you can swap windows, play/pause videos, control volume. 

KEEP IN MIND this is designed for MacOS, so the commands should be modified for Windows/Linux.

Mission Control:
Swipe up with the palm facing the camera to open Mission Control, Swipe down for vice versa.

Volume Control:
Hold two fingers up and move your hand up to increase the volume, move your hand down to decrease the volume.

Swapping Windows:
While the palm is facing the camera, swipe to the side you want to switch to(Swiping left slides to the window to the left and swiping right moves to the window in the right of the current window).

Play/Pause Media:
Open your palm and face it to the camera for a split second(If it is held too long, it will register as either Mission Control gesture, or Swap Window gesture based on where your hands move).
